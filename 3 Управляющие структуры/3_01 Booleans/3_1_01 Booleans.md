Booleans

Булевый тип в Ruby может принимать одно из двух значений: true или false.

sun_is_yellow = true
people_on_mars = false

                  
Булевый тип обычно используется в условиях и циклах.

Еще одно значение, с которым вы будете иногда встречаться это nil. Оно обозначает отсутствие какого-либо значения.
