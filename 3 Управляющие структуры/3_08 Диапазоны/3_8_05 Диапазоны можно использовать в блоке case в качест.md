Диапазоны можно использовать в блоке case в качестве значений оператора when.

Например:

age = 33

case age
when 0..18
  puts "Ребенок"
when 19..39
  puts "Молодой человек"
when 40..65
  puts "Средних лет"
else
  puts "Пожилой"
end

# выведет: Молодой человек

                  
Диапазоны также полезны для циклов for. Вы узнаете о них на следующем уроке.
