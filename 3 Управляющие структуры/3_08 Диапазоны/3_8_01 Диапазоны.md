Диапазоны (Ranges)

Диапазон представляет собой последовательность. Например, числа от 0 до 10, от 56 до 82, буквы от a до z - это все примеры диапазонов.

Для создания диапазонов в Ruby используются специальные операторы:

 .. (двоеточие),
 ... (троеточие).

Двоеточие создает диапазон включающий в себя и первое и последнее значение, а троеточие - диапазон, исключающий верхнее значение (не включает последнее значение).

Например:

a = (1..5).to_a
print a # выведет: [1, 2, 3, 4, 5]

b = (79...82).to_a
print b # выведет: [79, 80, 81]

c = ("a".."d").to_a
print c # выведет: ["a", "b", "c", "d"]

                  
Как можно увидеть на примере выше, при операторе .. в диапазон включается и начальное значение и последнее, а при операторе ... последнее не включается.
