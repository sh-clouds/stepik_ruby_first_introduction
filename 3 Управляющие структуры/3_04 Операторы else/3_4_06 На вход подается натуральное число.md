На вход подается натуральное число.

Если это число равно 6, нужно вывести слово Суббота, если число равно 7, нужно вывести слово Воскресенье. Если число меньше или равно 5, нужно вывести фразу Рабочий день. Во всех иных случаях нужно вывести фразу Неправильный день

Sample Input:
6

Sample Output:
Суббота

Напишите программу. Тестируется через stdin → stdout
Верно решили 738 учащихся
Из всех попыток 47% верных


# put your ruby code here
a = gets.to_i
if a == 6
    puts("Суббота")
elsif a == 7
    puts("Воскресенье")    
elsif a <= 5
    puts("Рабочий день")      
else
    puts("Неправильный день")    
end
