Оператор elsif

Конструкция elsif, это объединение else и if, как мы можем понять это из ее названия. Так же как и else, она расширяет возможности оператора if для выполнения разных действий в том случае, когда условие первого оператора if равняется false. Но, в отличии от else, исполнение кода после оператора происходит только тогда, когда условие оператора elsif будет равняться  true.

Например, следующий код принимает на вход два числа и выводит соответствующий текст, a больше b, или a равно b, или a меньше b:

a = gets.to_i
b = gets.to_i

if a > b
  puts "a больше чем b"
elsif a == b
  puts "a равно b"
else
  puts "a меньше чем b"
end

                  
В одном операторе if может быть несколько elsif.

Ниже приведен общий вид оператора if...elsif...else: 

if условие
  код будет выполнен если условие равно true
elsif условие2
  код будет выполнен если условие2 равно true
else
  код будет выполнен если оба условия равны false
end

                  
Помните, что последовательность составных элементов должна быть именно такой - один if, далее один или несколько elsif, и в конце (не обязательно) else.

В else никогда не указывайте условие, этот оператор выполняет следующий за ним код, если условия всех операторов перед ним будут равны false.

Когда первое условие в блоке условного оператора if (само условие if или условие в elsif ) равняется true, выполняется следующий за ним код и происходит выход из всего блока if.
