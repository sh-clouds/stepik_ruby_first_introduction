Для любителей "однострочников"

Если вам нужно проверить только одно условие, операторы if и unless можно использовать в одну строку.

Например:

age = 21

puts "Вам уже достаточно лет" if age >= 18

puts "Вам уже достаточно лет" unless age < 18

                  
В этом примере код перед if выполнится только в том случае, если условие будет вычислено как true.
Код перед unless, наоборот, выполнится только в том случае, если условие будет ложным false.
