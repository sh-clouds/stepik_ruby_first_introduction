Оператор или (||)

Оператор или (||) возвращает true, если один из операндов (или оба) истинны, и false, если оба операнда ложны.

Например:

doc = "Отсутствует"
age = 68
if doc == "Присутствует" || age >= 65
  puts "Пенсионер"
end

# выведет: Пенсионер

                  
Обратите внимание, что операторы (||) и (or) не являются одинаковыми и имеют разный приоритет вычисления.
