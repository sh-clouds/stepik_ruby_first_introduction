Логические операторы

Логические операторы используются для создания конструкций, которые проверяют более одного условия в условном операторе if (и не только).

В Ruby есть три логических оператора:

и (&&),
или (||),
не (!).
При использовании логического оператора и (&&) условие считается true тогда и только тогда, когда оба его операнда равны true. В противном случае все условие оценивается как false.

Например:

age = 33
if age >= 19 && age <= 39
  puts "Молодой человек"
end

# выведет: Молодой человек

                  
Ruby может использовать слова вместо символов логических операторов (and, or, not), но они имеют меньший приоритет и, как правило, их стараются избегать.
