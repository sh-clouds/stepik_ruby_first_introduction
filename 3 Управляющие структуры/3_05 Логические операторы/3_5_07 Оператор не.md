Оператор не (!)

Оператор не (!) возвращает противоположное логическое значение операнда (выражения).
Результат при не true будет false , а при не false - true.

x = 12
puts !(x > 10)

# выведет: false

                  
В этом коде x > 10 является true, а затем оператор не (!) меняет его значение на false.

Вы можете объединять несколько условий с помощью логических операторов для проверки множественных условий. С помощью скобок можно группировать отдельные условия для определения порядка выполнения операций и большей наглядности. Например:

(x > y && y < 100) || (x < y && y > 100)
