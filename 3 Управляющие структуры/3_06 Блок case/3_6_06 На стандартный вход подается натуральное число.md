На стандартный вход подается натуральное число.

Используя блок case, выведите в случае, если число будет равно 1, 2 или 3 - слово Плохо; если число равно 4 - выведите слово Хорошо; если число равно 5 - выведите слово Отлично. Во всех остальных случаях нужно вывести фразу Неправильная оценка

Sample Input:
2

Sample Output:
Плохо

Напишите программу. Тестируется через stdin → stdout
Верно решили 2 022 учащихся
Из всех попыток 53% верных



# put your ruby code here
case gets.to_i 
when 1, 2, 3
    puts "Плохо"
when 4
    puts "Хорошо"
when 5
    puts "Отлично"    
else
    puts "Неправильная оценка" 
end
