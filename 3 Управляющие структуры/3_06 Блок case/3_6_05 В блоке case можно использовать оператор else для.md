В блоке case можно использовать оператор else для выполнения кода в том случае, если не совпадают условия ни одного из операторов when:

age = 9

case age
when 1, 2, 3
  puts "Малыш"
when 4, 5, 6
  puts "Ребенок"
else
  puts "Уже не ребенок"
end

# выведет: Уже не ребенок

                  
Условный оператор if можно использовать в тех случаях, когда надо определить, верно ли то или иное условие, а блок case - в тех, когда нужно сделать выбор из нескольких вариантов, основываясь на одном значении.
