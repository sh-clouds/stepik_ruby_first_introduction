Условные операторы if могут также быть вложенными, один внутри другого.

При этом нужно учитывать, что внутренний if является частью внешнего. Это один из способов убедиться в выполнении сразу нескольких условий. Имейте в виду, что если первое условие if не выполняется, то прекращается работа всего блока.

Например:

num = 10
if num > 5
  puts "Число больше 5"
  if num < 20
    puts "Число между 5 и 20"
  end
end

# выведет:
# Число больше 5
# Число между 5 и 20

                  
И не забывайте, что каждому if обязательно должен соответствовать свой end.
