Условный оператор if

Вы можете использовать условный оператор if для выполнения кода в том случае, когда выполняется определенное условие. Если условное выражение оценивается как true, то выполняется код в блоке if. В противном случае код игнорируется.

Синтаксис блока условного оператора в общем виде выглядит так:

if условие 
  код будет выполнен если условие равно true
end

                  
Например:

age = 66

if age > 65
  puts "Вы пенсионер"
end

# выведет: Вы пенсионер

                  
Оценивается условие age > 65. Если оно верно (то есть равно true), то выполняется выражение внутри блока if и программа выводит Вы пенсионер. Внутри одного оператора if может быть несколько выражений:

if age > 65
  puts "Вы пенсионер"
  puts "И получаете пенсию. Большую"
end

                  
Условный оператор начинается с ключевого слова if и обязательно заканчивается ключевым словом end
