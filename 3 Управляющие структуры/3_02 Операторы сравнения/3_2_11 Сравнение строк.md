Сравнение строк

Все операторы сравнения также можно использовать для лексикографического сравнения строк (алфавитный порядок слов основывается на алфавитном порядке составляющих их букв).

Например:

puts "abc" < "bca" # выведет: true

                  
Условно представьте себе, что каждая буква алфавита имеет вес в виде порядкового номера этой буквы в алфавите. Так b (2) будет меньше e (5). При сравнении строк не имеет значения их длина, сравнение происходит поочередно по буквам, пока не будет определена бОльшая буква.

puts "abc" > "abd" # выведет: false

                  
Так, в примере выше сначала сравниваются первые буквы (a). Они одинаковы. Значит далее сравниваются вторые буквы (b). Они тоже одинаковы. Далее сравниваются третьи буквы и тут d больше с. Даже если дальше буду идти еще буквы - сравнение будет остановлено и выдан результат.
