Следующие операторы сравнения это:

- больше > и меньше < 
- больше или равно >= и меньше или равно <=

Например:

puts 9 > 5    # выведет: true
puts 8 < 3    # выведет: false
puts 2 >= 2.0 # выведет: true
puts 4 <= 7   # выведет: true

                  
Обратите внимание на третий пример, где сравнивается целое число и вещественное число. Обычные операторы сравнения сравнивают фактическое значение, но не учитывают тип. Поэтому в этом примере значения равны.
