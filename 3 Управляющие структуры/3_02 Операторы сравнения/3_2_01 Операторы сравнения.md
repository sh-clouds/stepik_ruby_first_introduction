Операторы сравнения 

Операторы сравнения сравнивают два значения. Обычно это числа или строки. При этом нельзя сравнивать число со строкой.
Операторы сравнения используются внутри условных операторов и в результате выдают значения true или false.

Например:

x = 11
y = 3
puts x == y      # выведет: false
puts 3 == y      # выведет: true
puts x - 8 == y  # выведет: true

                  
Сравнение с помощью оператора равно == возвращает true, если два операнда (значения или выражения) равны, и false, если они не равны.

Будьте осторожны, чтобы не перепутать оператор присвоения = с оператором сравнения ==.
