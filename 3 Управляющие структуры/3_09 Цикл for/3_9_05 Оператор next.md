Оператор next

Оператор next может быть использован для того, чтобы пропустить одну итерацию цикла и продолжить со следующей (аналог continue в других языках программирования).

Например:

for i in 0..8
  if i % 2 == 0
    next
  end
  puts i
end

# выведет:
# 1
# 3
# 5
# 7

                  
При этом будут выведены только нечетные числа от 0 до 8, потому что итерации цикла с четными значениями переменной i будут пропущены. 

В Ruby также есть оператор redo, который вызывает повторную итерацию текущего цикла.
А оператор retry заставляет весь цикл начинаться заново с начала.
