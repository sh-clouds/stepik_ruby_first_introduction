Цикл for

Цикл for используется для перебора (и совершения над ними каких-то действий) определенного набора значения (диапазона или массива).

Общий вид цикла for:

for элемент in диапазон/массив
  выполняемый код (тело цикла)
end

                  
элемент: это переменная цикла, которой присваивается значение первого элемента диапазона (или массива). Каждую последующую итерацию цикла ей присваивается следующее значение диапазона (или массива_, и так до самого конца диапазона (массива);
диапазон/массива: это диапазон или массив, элементы которого будут перебирать по очереди, путем присваивания их значений переменной.

Например:

for i in 1..5
  puts i
end

# выведет:
# 1
# 2
# 3
# 4
# 5

                  
Код выше выведет числа от 1 до 5.

В начале выполнения цикла переменной i будет присвоено первое значение диапазона, то есть 1. В следующей итерации этой переменной будет присвоено значение 2 и так далее до конца диапазона.

Таким образом , в цикле for для каждого элемента диапазона тело цикла выполняется один раз.

Не забывайте, что блок цикла должен завершаться ключевым словом end.
