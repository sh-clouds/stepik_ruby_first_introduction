Оператор break

Оператор break используется для остановки цикла (любого, не только for) и выхода из него.

Пример прерывания цикла for:

for i in 1..5
  if i > 3
    break
  end
  puts i
end

# выведет:
# 1
# 2
# 3

                  
Цикл прерывается при выполнении условия i > 3.

Пример прерывания цикла while:

x = 10
while x < 14
  if x == 12
    break
  end
  puts x
  x += 1
end

# выведет:
# 10
# 11

                  
Цикл прерывается когда выполнится условие x == 12.
