На стандартный вход подается целое число. Необходимо вывести все нечетные числа от нуля по это число включительно.

Sample Input:
7

Sample Output:
1
3
5
7

Напишите программу. Тестируется через stdin → stdout
Верно решил 1 791 учащийся
Из всех попыток 40% верных


# put your ruby code here
a = gets.to_i
i = 0
loop do
    if i > a 
        break
    end
    if i%2==0
        i+=1
        next
    end
    puts i
    i+=1
end
