Цикл loop do

Еще один оператор цикла в Руби - это оператор loop do. Он может понадобиться в ситуации, когда вам надо, чтобы цикл выполнился хотя бы один раз при любых условиях.

Этот цикл не имеет своих условий, то есть является бесконечным. Выйти из него можно с помощью использования оператора break.

Например:

x = 1
loop do
  puts x
  x += 1
  if x > 5
    break
  end
end

# выведет:
# 1
# 2
# 3
# 4
# 5

                  
При этом будут выведены числа от 1 до 5. Когда выполнится условие x > 5, цикл прекратит выполнение.

Обратите внимание, что если мы не включим в цикл условие и оператор break, то цикл будет выполняться вечно.

И не забывайте, что блок цикла должен завершаться ключевым словом end.
