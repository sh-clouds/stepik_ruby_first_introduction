Цикл while

Для того чтобы выполнить несколько раз подряд один и тот же код используются циклы. В Ruby есть три разновидности циклов - while, until и for. Рассмотрим их по порядку.

В цикле while блок кода выполняется до тех пор, пока указанное условие истинно (true). Данный вид цикла удобно использовать, когда вы не знаете сколько именно раз нужно выполнить цикл, но знаете условие, при котором его работу нужно прекратить.

Общий вид цикла:

while условие равно true 
   код будет выполняться (тело цикла)
end

                  
Блок цикла обязательно должен завершаться ключевым словом end

Например:

x = 1
while x <= 5
  puts x
  x += 1
end

# выведет в столбик числа от 1 до 5

                  
При выполнении кода выше будут выведены числа от 1 до 5. После того, как переменная x станет больше 5, условие x <= 5 становится ложным (false), и цикл завершает свое выполнение.

Обратите внимание на то, что в случае, если условие никогда не станет равно false, цикл будет выполняться бесконечно и программа "зависнет". Поэтому мы обязательно должны предусмотреть в теле цикла изменение переменной из условия таким образом, чтобы оно в итоге стало равно false. Как в примере выше, например.
