Методы getter

Также можно создать метод экземпляра для доступа к переменным экземпляра из-за пределов объекта.
Например, если мы хотим получить доступ к переменной экземпляра @color объекта Car, нам нужен метод, который возвращает значение этой переменной:

class Car
  def initialize(color)
    @color = color
  end
  def get_color
    @color
  end
end

audi = Car.new("Красный")
puts audi.get_color

# выведет: Красный

                  
В примере выше мы создали метод экземпляра get_color, который возвращает значение переменной экземпляра @color, а затем вызвали его для нашего объекта audi.

Методы getter и setter называются средствами доступа (accessors).
Метод, который используется для получения значения переменной, называется методом getter (в примере выше это get_color).
Метод, который используется для изменения значения переменной, называется методом setter (о нем мы поговорим на следующем теоретическом шаге).
