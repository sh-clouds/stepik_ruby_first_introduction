Методы setter

Для доступа к переменной экземпляра используются методы getter. Но если мы захотим изменить значение переменных экземпляра, нам понадобятся методы setter.
В Ruby есть специальный синтаксис для определения методов setter: за именем метода следует знак равно =:

class Car
  def initialize(color)
    @color = color
  end
  def get_color
    @color
  end
  def set_color=(color)
    @color = color
  end
end

bmw = Car.new("Красный")
bmw.set_color = "Синий"
puts bmw.get_color

# выведет: Синий

                  
В примере выше set_color - это метод setter, который устанавливает значение переменной экземпляра @color  в значение его параметра color.

Обратите внимание на специальный синтаксис, используемый при вызове метода: bmw.set_color = "Синий".
Обычно для вызова метода setter используется конструкция вида bmw.set_color=("Синий"), где set_color= это имя метода, а строка Синий - это аргумент, передаваемый в метод. Однако, для методов setter Ruby позволяет использовать более естественный синтаксис присваивания, как в нашем примере выше: bmw.set_color = "Синий".
