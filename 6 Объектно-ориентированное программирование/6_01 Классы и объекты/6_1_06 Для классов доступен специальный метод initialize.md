Для классов доступен специальный метод initialize, который вызывается при создании объекта. То есть он выполняется в момент создания экземпляра (объекта) класса.

Он определяется внутри класса так же, как и любой другой метод класса:

class Car
  def initialize
    puts "Др-др-др"
  end
end

                  
Целью метода initialize является инициализация переменных класса для нового объекта. Например, при создании объекта Car метод initialize может установить количество шин в значение 4).

Метод initialize в других объектно-ориентированных языках программирования известен как constructor.
