super

В Ruby есть ключевое слово super, которое используется для вызова методов из суперкласса в подклассе. Например:

class Automobile
  def info
    puts "Это автомобиль"
  end
end

class Truck < Automobile
  def info
    super
    puts "Грузовой"
  end
end

                  
super вызывает метод info из суперкласса Automobile. При этом в подклассе в этот метод добавляется вывод еще одной строки. И теперь, если мы создадим объект класса Truck и вызовем его метод info, то получим следующее:

Man = Truck.new
Man.info
# выведет: 
# Это автомобиль
# Грузовой

                  
Использование ключевого слова super позволяет удалять дублирующий код, расширяя при необходимости поведение методов суперкласса в подклассах.
