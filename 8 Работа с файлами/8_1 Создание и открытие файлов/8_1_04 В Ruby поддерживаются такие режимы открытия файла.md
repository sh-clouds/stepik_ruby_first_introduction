В Ruby поддерживаются такие режимы открытия файла:

r файл открывается только для чтения. Курсор помещается в начало файла перед первым символом (этот режим используется по умолчанию);
r+ файл открывается для чтения и записи. Курсор помещается в начало файла перед первым символом;
w файл открывается только для записи. Файл будет полностью перезаписан новой информацией. Если файл не существует, он будет создан;
w+ файл открывается для чтения и записи. Файл будет полностью перезаписан новой информацией. Если файл не существует, он будет создан;
a файл открывается только для записи. Курсор помещается в конец файла, если файл существует, в противном случае создается новый файл для записи. Информация дописывается после уже существующей в файле;
a+ файл открывается для чтения и записи. Курсор помещается в конец файла, если файл существует, в противном случае создается новый файл для записи. Информация дописывается после уже существующей в файле.

Если файл открывается в режиме только для чтения, его режим не может быть изменен на режим для записи в процессе работы с файлом. Точно так же в открытом файле режим не может быть изменен с режима только для записи на режим для чтения.
