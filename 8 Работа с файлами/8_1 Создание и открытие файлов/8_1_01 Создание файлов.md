Создание файлов

До сих пор мы выполняли задания связанные с вводом и выводом используя стандартные потоки ввода/вывода (stdin/stdout) и, соответственно, методы gets и puts. Однако более привычной формой ввода и вывода являются файлы.

Ruby позволяет нам легко работать с файлами, используя встроенный класс File:

file = File.new("sample.txt", "w+")

                  
Код выше создает файл под названием sample.txt с режимом w+ для доступа на чтение и запись. При этом файл будет создан в том же каталоге, где находится программа. Кроме этого, создается объект для работы с этим файлом под названием file. Теперь мы можем использовать этот объект для выполнения различных операций с файлом.

После использования файла, объект должен быть закрыт с использованием метода close:

file.close

                  
Хотя это не обязательно, но привыкайте закрывать файлы после того, как поработали с ними. Это позволит избежать разного рода проблем.
