Также можно проверить, является ли файл записываемым, читаемым или исполняемым:

file = File.new("sample.txt", "w+")
file.puts("Текст")
file.close

puts File.readable?("sample.txt")   # true
puts File.writable?("sample.txt")   # true
puts File.executable?("sample.txt") # false

                  
Более подробно о классе File и о его методах можно почитать по ссылке здесь (https://ru.wikibooks.org/wiki/Ruby/%D0%A1%D0%BF%D1%80%D0%B0%D0%B2%D0%BE%D1%87%D0%BD%D0%B8%D0%BA/File#%D0%9A%D0%BB%D0%B0%D1%81%D1%81_File) или здесь (https://ruby-doc.org/3.2.2/File.html).
