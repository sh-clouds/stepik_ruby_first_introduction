Получение информации о файлах

В Ruby имеются методы для получения полезной информации о файле. Например, метод size возвращает размер файла.
А метод zero? возвращает true, если именованный файл существует и имеет нулевой размер (является пустым).
Например:

file = File.new("sample.txt", "w+")
file.puts("Текст")

puts file.size 
# выведет: 19

file.close

puts File.zero?("sample.txt") 
# выведет: false
