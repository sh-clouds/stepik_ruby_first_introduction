Работа с файлами

При работе с файлами важно сначала проверить, существует ли файл, чтобы предотвратить ошибку, например, при попытке его открыть или удалить.

Метод file? предоставляет простой способ проверки существования файла. Например:

File.open("sample.txt") if File.file?("sample.txt")

                  
Код выше проверяет, существует ли файл перед тем, как попробовать его открыть.

File.file? возвращает true, если файл существует, и false, если он не существует.
