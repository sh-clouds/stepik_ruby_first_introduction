Чтение файлов

Для чтения содержимого всего файла можно использовать метод File.read. Например:

file = File.new("sample.txt", "w+")
file.puts("Хороший день для изучения Ruby")
file.puts("Сегодня я точно смогу узнать много нового")
file.close

puts File.read("sample.txt")

# выведет:
# Хороший день для изучения Ruby
# Сегодня я точно смогу узнать много нового
