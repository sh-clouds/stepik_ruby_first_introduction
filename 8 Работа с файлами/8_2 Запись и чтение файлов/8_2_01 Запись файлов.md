Запись файлов

Для записи содержимого в файл могут быть использованы методы puts и write . Разница между ними в том, что при использовании puts в конец строки добавляется перенос строки, а при использовании метода write - нет.
Например:

file = File.new("sample.txt", "w")
file.puts("Очень важный текст")
file.write("Не менее важный текст")
file.close

                  
Не забудьте закрыть файл (используя метод close) после выполнения всех операций с файлом.
