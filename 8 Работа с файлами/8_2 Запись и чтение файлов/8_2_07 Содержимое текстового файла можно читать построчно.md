Содержимое текстового файла можно читать построчно, используя метод readlines. Например:

File.open("sample.txt", "a+") {
  |file| file.puts("Первая строка текста")
  file.puts("Вторая строка текста")
  file.puts("Третья строка текста")
}

File.readlines("sample.txt").each {
  |line| puts " - #{line}"
}

# выведет:
# - Первая строка текста
# - Вторая строка текста
# - Третья строка текста

                  
Метод readlines считывает весь файл по одной строке и возвращает эти строки в массиве.
