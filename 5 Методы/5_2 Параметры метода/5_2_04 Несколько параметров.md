Несколько параметров

В методе можно указывать несколько параметров (их может быть столько, сколько вам нужно), разделенных запятыми:

def sum(a, b)
  puts a + b
end

sum(13, 21) 
# выведет: 34

sum(3, 14)
# выведет: 17

                  
В качестве аргументов при вызове метода можно использовать не только значения, но и переменные:

def sum(a, b, c)
  puts a + b + c
end

x = 3
y = 21
z = 10
sum(x, y, z)
# выведет: 34

                  
Обратите внимание, что количество аргументов при вызове метода должно соответствовать количеству параметров в его описании
