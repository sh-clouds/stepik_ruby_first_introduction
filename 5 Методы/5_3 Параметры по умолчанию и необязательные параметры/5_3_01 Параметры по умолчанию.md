Параметры по умолчанию

Для ситуаций, когда аргументов может оказаться меньше чем параметров, можно в методе для параметров установить значения по умолчанию. В такой ситуации метод все равно будет работать, даже если в него будут переданы не все аргументы.
Например:

def sum(x, y = 5)
  puts x + y
end

                  
В данном примере мы создали метод с двумя параметрами, но для параметра y мы указали значение по умолчанию равное 5.
И теперь, если мы при вызове метода передадим в него только один аргумент, то метод просто будет в качестве второго аргумента использовать значение 5:

f = 11
sum(f)
# выведет: 16

                  
При этом мы можем передавать и два аргумента, и метод тоже будет нормально работать. Значение по умолчанию вступает в игру только тогда, когда значение для этого параметра не передается.
