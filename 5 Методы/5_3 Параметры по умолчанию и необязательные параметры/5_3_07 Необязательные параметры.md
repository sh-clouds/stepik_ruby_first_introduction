Необязательные параметры

Мы также можем создавать методы, которые могут принимать любое количество аргументов. Для этого используется следующий синтаксис:

def simple_method(*m)
  puts m
end

simple_method(34, "Stepik", true, 18)

                  
И теперь в метод можно передавать любое количество аргументов. В коде выше m - это массив. Когда мы передаем аргументы в метод, они становятся элементами массива m.

При этом, если вызвать метод без аргументов, то массив m будет пустым.

Такой подход позволяет, например, задавать для метода необязательные параметры:

def another_method(a, b, *c)
  # какой-то код
end

                  
another_method имеет два обязательных параметра и один необязательный. Поэтому в метод можно передать два и более аргументов. При этом необязательный параметр должен идти последним.
