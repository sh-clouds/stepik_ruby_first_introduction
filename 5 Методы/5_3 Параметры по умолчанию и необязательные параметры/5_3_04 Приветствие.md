Приветствие

Рассмотрим еще один пример использования параметров по умолчанию. Давайте создадим программу, которая может приветствовать пользователя:

def greet(name = "")
  if name == ""
    puts "Приветствую!"
  else
    puts "Добро пожаловать, #{name}!"
  end
end

greet()
# выведет: Приветствую!

greet("Михаэль")
# выведет: Добро пожаловать, Михаэль!

greet(gets.chomp)

                  
Мы определили метод под названием greet, который принимает один параметр со значением по умолчанию и выводит приветственное сообщение на основе значения этого параметра.

При первом вызове метода мы ничего не передаем в качестве аргумента и получаем обычное приветствие. 
При втором вызове мы передаем в качестве аргумента имя и оно используется, чтобы выдать нам персонифицированное приветствие.
Третий вариант позволяет нам использовать интерактивность. В этом случае в качестве аргумента метода будет использоваться пользовательский ввод.
