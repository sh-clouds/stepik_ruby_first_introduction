Возврат без return

Даже если вы явно не включаете оператор return в свой метод, Ruby всегда возвращает результат работы последней строки тела метода. Например:

def example(a, b)
  a = b - 2
  b = a - 3
end

puts example(5, 6)
# выведет: 1

                  
Приведенный в примере метод вернет результат своего последнего выражения, который равен b = a - 3.
