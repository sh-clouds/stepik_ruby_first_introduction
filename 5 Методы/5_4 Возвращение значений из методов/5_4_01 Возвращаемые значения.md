Возвращаемые значения

Все методы, которые мы до сих пор определяли в текущем модуле, просто выводили результаты своей работы. Однако обычно метод используется для выполнения необходимых действий и возврата полученного результата, чтобы возвращаемое значение можно было использовать далее в программе (например, присваивая переменной возвращаемое значение).

Для возврата значений из метода используется ключевое слово return. Например:

def sum(x, y)
  res = x + y
  return res
end

s = sum(11, 3)
puts s
# выведет: 14

                  
Определенный выше метод sum возвращает сумму своих параметров, которая затем присваивается переменной s. И затем эту переменную мы можем использовать далее в программе.

Обратите внимание, что любой код в методе после оператора return не будет выполнен, так как метод прекращает выполнение сразу после выполнения оператора возврата. Например:

def example(a)
  a = 5
  return a
  a = 9
end

                  
Метод, описанный выше, вернет значение 5. Выражение a = 9 не будет выполнено, так как оно идет после выполнения оператора возврата.
