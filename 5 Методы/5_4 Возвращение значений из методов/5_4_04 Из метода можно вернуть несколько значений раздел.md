Из метода можно вернуть несколько значений, разделив их запятыми в операторе возврата. Например:

def squares(a, b, c)
  return a * a, b * b, c * c
end

arr = squares(1, 5, 2)
puts arr

# выведет
# 1
# 25
# 4

                  
В результате получается массив, содержащий все возвращаемые значения в виде отдельных элементов:

print arr
# выведет [1, 25, 4]
