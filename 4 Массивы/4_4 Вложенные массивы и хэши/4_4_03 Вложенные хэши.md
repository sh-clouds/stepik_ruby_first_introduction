Вложенные хэши

Так же как и массивы, вложенными могут быть и хэши:

cars = {
  bmw: { year:2016, color:"белый" },
  mercedes: { year:2012, color:"черный" },
  porsche: { year:2014, color:"синий" }
}

puts cars[:mercedes][:year]
# выведет: 2012

                  
Хэши и массивы могут иметь любой уровень вложенности, но имейте в виду, что чем больше будет уровней вложенности, тем менее удобно будет с ними взаимодействовать. Не стоит этим злоупотреблять.
