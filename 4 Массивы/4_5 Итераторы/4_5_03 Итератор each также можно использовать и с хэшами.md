Итератор each также можно использовать и с хэшами:

cars = { audi:1909, ford:1903, toyota:1937 }

cars.each do |key, value|
  puts "Компания #{key} создана в #{value} году"
end

# выведет:
# Компания audi создана в 1909 году
# Компания ford создана в 1903 году
# Компания toyota создана в 1937 году

                  
В приведенном выше примере ключом и значением являются переменные, которым при каждой итерации присваиваются соответствующие значения хэш-элементов. Вы можете использовать любое имя для этих переменных.

Ключевые слова do и end указывают на начало и конец блока кода в Ruby. После открытия блока у нас есть параметры блока внутри вертикальных черт  | | 

Также Ruby предоставляет короткий способ написания блоков: можно использовать фигурные скобки для начала и конца блоков кода (вместо ключевых слов do/end). Таким образом, приведенный выше пример можно написать так:

cars = { audi:1909, ford:1903, toyota:1937 }

cars.each { |key, value| puts "Компания #{key} создана в #{value} году" }

                  
Итератор each может также использоваться в диапазонах.
А для строк можно использовать итератор each_char для перебора всех символов строки.

Примеры:

sum = 0
(1..10).each do |x|
  sum += x
end
puts sum 
# выведет сумму чисел от 1 до 10, то есть число 55

"строка".each_char do |x|
  puts x
end
# выведет все символы строки, каждый с новой строки
