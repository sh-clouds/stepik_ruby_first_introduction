Методы для удаление элементов

Также существуют методы и для удаления элементов из массивов. Это методы pop и delete_at.

Метод pop удаляет последний элемент массива.

Например:

cities = ["Рим", "Лондон", "Осло"]
cities.pop
puts cities
# выведет: 
# Рим
# Лондон

                  
Чтобы удалить конкретный элемент массива, можно использовать метод delete_at.

Например:

cities = ["Рим", "Лондон", "Осло"]
cities.delete_at(1)
puts cities
# выведет: 
# Рим
# Осло

                  
Надеюсь, вы помните, что индекс начинает отсчет с 0. Значит, второй элемент в массиве будет иметь индекс 1. Поэтому в примере выше мы указали индекс 1 для удаления. Это второй элемент массива, который и был из него удален.
