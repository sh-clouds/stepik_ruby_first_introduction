Методы для добавления элементов

В качестве альтернативы можно использовать методы push и insert (подробнее о методах мы узнаем в следующем модуле. А пока просто помните, что метод - это код, который выполняет какое-то действие).

Например, можем заменить вторую строку предыдущего примера следующей строкой:

cities.push("Пекин")

                  
И получим точно такой же результат. В конец массива будет добавлен элемент Пекин.

А вот метод insert позволяет вставлять элемент в конкретное место массива:

cities = ["Рим", "Лондон", "Осло"]
cities.insert(1, "Пекин")
puts cities
# выведет: 
# Рим
# Пекин
# Лондон
# Осло

                  
В примере выше мы вставляем элемент со значением Пекин на место массива с индексом 1.

При выводе массива на экран, метод puts выводит по одному элементу на строку, в то время как метод print просто выводит скобки и список всех элементов:

cities = ["Рим", "Лондон", "Осло"]
print cities
# выведет: ["Рим", "Лондон", "Осло"]
