Добавление элементов

Массив одновременно может содержать различные типы элементов.

Например:

diff = ["Норвегия", 11, true, 3.14]

puts diff[0]  # выведет: Норвегия
puts diff[1]  # выведет: 11
puts diff[-1] # выведет: 3.14

                  
Для добавления в массив новых элементов можно использовать оператор <<:

cities = ["Рим", "Лондон", "Осло"]
cities << "Пекин"
puts cities

# выведет: 
# Рим
# Лондон
# Осло
# Пекин

                  
При этом в конец массива будет добавлен элемент со значением Пекин.
