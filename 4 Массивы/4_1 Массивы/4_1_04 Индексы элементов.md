Индексы элементов

Каждый элемент массива имеет свой уникальный индекс. Нумерация массивов в Ruby начинается с нуля. С помощью индексов можно обращаться к элементам массива. Для этого нужно указать индекс нужного элемента в квадратных скобках после имени массива.

Например:

items = ["Lenovo", "Asus", "Sony", "Samsung"]
puts items[0]
# выведет: Lenovo

                  
Используя индекс можно любому элементу массива присвоить новое значение:

items = ["Lenovo", "Asus", "Sony", "Samsung"]
items[2] = "Apple"
# после этого массив будет иметь следующий вид:
# items = ["Lenovo", "Asus", "Apple", "Samsung"]

                  
В примере выше элементу массива с индексом 2 будет присвоено значение Apple.

Можно использовать отрицательный индекс. Он позволяет отсчитывать элементы с конца массива. Например, индекс -1 указывает на последний элемент массива, -2 - на предпоследний и так далее:

items = ["Lenovo", "Asus", "Sony", "Samsung"]
puts items[-1]
# выведет: Samsung
