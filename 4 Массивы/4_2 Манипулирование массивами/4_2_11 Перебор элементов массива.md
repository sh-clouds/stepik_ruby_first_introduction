Перебор элементов массива

Одним из способов прохождения по массиву элементов является использование цикла for для этого.

Например:

city1 = ["Пекин", "Токио", "Дели"]
for x in city1
  puts "Город: #{x}"
end
# выведет: 
# Город: Пекин
# Город: Токио
# Город: Дели

                  
В данном примере все элементы массива будут по порядку обработаны циклом и выведены по одному на строку.
