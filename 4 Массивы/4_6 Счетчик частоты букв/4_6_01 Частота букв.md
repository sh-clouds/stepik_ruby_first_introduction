Частота букв

Создадим программу, которая будет подсчитывать частоту букв (количество вхождений) в заданной строке.
Сначала нам понадобится строка:

text = "Мне нравится изучать Ruby на Степике!"

text.downcase!

                  
Метод downcase! используется для преобразования всех букв в строке в нижний регистр.

Далее нам понадобится хэш, чтобы сохранить буквы в качестве ключей, а количество букв в качестве соответствующих значений. Для этого нужно создать пустой хэш и установить значение по умолчанию для всех значений равным 0.

freqs = {}
freqs.default = 0

                  
Метод default используется для установки значения по умолчанию для ключей хэша. То есть любой ключ, которому еще не было присвоено какое-нибудь значение, будет иметь вот это значение по умолчанию.
