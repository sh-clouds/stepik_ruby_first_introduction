Лямбды (lambda)

Lambda - это разновидность Proc. Лямбда на самом деле является экземпляром класса Proc.
Для создания lambda в Ruby используется следующий синтаксис:

talk = lambda { puts "Привет!" }

                  
В качестве альтернативы можно использовать следующий синтаксис:

talk = ->() { puts "Привет!" }

                  
Как и в случае с Proc, для выполнения lambda используется метод call :

talk.call

                  
В других языках программирования lambda обычно называется анонимной функцией.
