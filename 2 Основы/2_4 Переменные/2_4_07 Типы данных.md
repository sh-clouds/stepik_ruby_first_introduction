Типы данных

Все переменные в Ruby могут содержать любые типы данных. Ruby автоматически определяет тип данных по их значению.

Например:

x = 27        # целое число
y = 3.14      # вещественное число
s = "Степик"  # строка
b = true      # булевый тип

                  
Вещественное число (или число с плавающей точкой) состоит из целой части, точки и дробной части. Разделителем служит исключительно точка. 

При этом в любое время вы можете присвоить переменной значение другого типа.

Однако хорошей практикой считается использование одного и того же типа данных в одной переменной во всей программе. Это помогает избежать ошибок.
