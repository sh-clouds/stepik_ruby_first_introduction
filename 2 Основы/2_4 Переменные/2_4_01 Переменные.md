Переменные

Переменные используются для хранения определенной информации. Это могут быть числа, строки, массивы и др. Каждая переменная имеет свое уникальное (обычно) имя. Давайте посмотрим на пример:

x = 27

                  
Здесь x - это имя переменной, 27 - её значение. А знак равенства (=) используется для того, чтобы присвоить переменной какое-то значение. В данном примере мы присвоили значение 27 переменной с именем x.

Знак равенства называется оператором присваивания.

Из самого названия "переменные" можно понять, что они могут изменяться. То есть мы можем одной и той же переменной присваивать разные значения, например:

x = 27
x = 11

                  
После выполнения этого кода, в переменной х будет храниться уже значение 11, а не 27.

Обращаясь к переменной по ее имени, мы можем получить доступ к ее значению. Например, мы можем использовать puts или print для вывода значения переменной:

x = 27
puts x # выведет 27

                  
Обратите внимание, что имена переменных могут состоять из латинских букв, цифр и символа подчеркивания (_), но не могут начинаться с большой буквы или цифры. Также в Ruby есть ряд ключевых слов, которые нельзя использовать для имен переменных.

Имя переменной может содержать кириллические символы, но не стоит этим злоупотреблять. Привыкайте использовать для этого латиницу.
