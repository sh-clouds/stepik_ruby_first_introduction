Ввод

Для получения информации от пользователя в Ruby можно использовать метод gets, который возвращает то, что пользователь вводит в виде строки. Чтобы использовать полученные входные данные в дальнейшем, можно присвоить возвращаемое значение переменной.

Например:

x = gets
puts x

                  
gets получает строку текста, включая новую строку в конце. Если вы не хотите включать новую строку, используйте метод gets.chomp:

puts "Введите свое имя"
name = gets.chomp
puts "Добро пожаловать, #{name}"

                  
Значение входных данных по умолчанию - строка. Для преобразования её в целое число можно использовать метод gets.to_i, а в вещественное - метод gets.to_f
