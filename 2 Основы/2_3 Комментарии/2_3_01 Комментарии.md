Комментарии
Комментарии - это строки примечаний в коде Ruby, которые игнорируются во время выполнения программы.
Символ хэштэга (#) используется в Ruby для создания однострочного комментария.

Например:

# это строка комментария, которая игнорируется во время выполнения программы
puts "Где-то выше скрывается комментарий"

                  
Все, что следует за символом хэштэга в этой строке, игнорируется при запуске программы.
