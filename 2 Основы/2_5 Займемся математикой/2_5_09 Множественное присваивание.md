Множественное присваивание

Ruby также поддерживает множественное присваивание переменных. Это позволяет инициализировать несколько переменных одной строкой кода.

Например, мы хотим объявить и инициализировать три переменные:

a = 5
b = 7
c = 9

                  
альтернативой этому коду может быть множественное присваивание:

a, b, c = 5, 7, 9

                  
Множественное присваивание также можно использовать для перестановки значений, содержащихся в двух переменных без использования третьей переменной:

a, b = b, a
